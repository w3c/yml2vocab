Class MiniDOM

A thin layer on top of the regular DOM Document. Necessary to "hide" the differences between the JSDOM and Deno's DOM WASM implementations; higher layers should not depend on these.

The class also includes some handy shorthands to make the code cleanerâ€¦

2024-02-02: as of today, with deno v. >1.4, this layer is not necessary any more, because jsdom finally runs with deno as well. The class has been kept as a separate layer following the wisdom of "ain't broken, don't fix it" :-)
